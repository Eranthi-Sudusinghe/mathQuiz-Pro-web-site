# mathQuiz-Pro-web-site

Math Quiz Pro is simply an online quiz app to test your knowledge of math and provides users with the ability to answer questions by displaying them in Quiz format.

 Features

- Interactive math quiz
- Score calculation
- Leaderboard display
- Responsive design utilizing Bootstrap
- Simple and user-friendly interface

 Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Google fonts

## Directory Structure

MathQuizPro/
│
├── index.html        ( The main web page)
├── styles.css        ( Application style)
├── script.js         ( Quiz logic and functionality)
└── README.md         ( Project descriptions)

 Pages

 Home Page
The home page provides the application to get started in conducting a quiz.

 Quiz Page
This page displays a Math question and allows a user to provide an answer.

 Leaderboard Page
This page displays the highest scores for each USER.

 How the App Works

1. User goes to website.
2. Clicks on **Start Quiz**.
3. Randomly generates a Math question.
4. User selects or inputs the answer.
5. System verifies answer and re-calculates score.
6. Displays the final score after completing the Quiz.

 How to Run the App

1. Download/clone the repository followed by accessing the project folder.
2. Run **index.html** by double clicking or using a web browser.

No server setup is required.
